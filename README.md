# 📱 Mobile Sales Analysis — Power BI Dashboard

An interactive Power BI dashboard built to analyze mobile sales 
transactions across major Indian cities, covering multiple brands, 
models, and customer demographics.

## 📊 Dataset Overview

| Column | Description |
|---|---|
| Transaction ID | Unique sale record |
| Day / Month / Year | Date of transaction |
| Brand | Apple, Samsung, Xiaomi, Vivo, OnePlus |
| Mobile Model | Specific model sold |
| Units Sold | Quantity per transaction |
| Price Per Unit | Sale price in ₹ |
| Customer Name & Age | Buyer details |
| City | Indian cities (Delhi, Mumbai, etc.) |
| Payment Method | UPI, Cash, Credit/Debit Card |
| Customer Ratings | Rating out of 5 |

## 🔍 Key Insights Covered

- Brand-wise sales and revenue comparison
- City-wise sales performance across India
- Payment method distribution (UPI, Cash, Cards)
- Customer age group and rating analysis
- Monthly and daily sales trends
- Top performing mobile models

1. Download and install Power BI Desktop (free)
2. Open `Mobile_Sales_Data.pbix` in Power BI Desktop
3. Explore the interactive dashboard with filters and slicers
